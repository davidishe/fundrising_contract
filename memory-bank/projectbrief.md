# PROJECT BRIEF: TON Crowdfunding System

## CORE OBJECTIVE

Создать простую систему краудфандинга на TON блокчейне:
- Пользователи отправляют TON → получают jetton
- Админ устанавливает и меняет цену вручную  
- Refund по исходной цене покупки
- Масштабируемость через индивидуальные пользовательские контракты

## ОСНОВНЫЕ ТРЕБОВАНИЯ

### 1. Инвестиционный Flow
- Пользователь отправляет TON
- Система рассчитывает количество jetton по текущей цене
- Выдает jetton пользователю
- Сохраняет данные об инвестиции (сумма + цена на момент покупки)

### 2. Refund Flow  
- Пользователь может вернуть jetton
- Система рассчитывает возврат по цене на момент покупки
- Сжигает jetton, возвращает TON

### 3. Админ Управление
- Установка цены jetton в ручном режиме
- Изменение цены в любое время
- Контроль системы

### 4. Архитектура Масштабирования
- Master контракт - основная логика
- Отдельный контракт для каждого пользователя
- Хранение истории инвестиций в пользовательских контрактах

## ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

### Smart Contracts
1. **Master Contract** - основная логика, управление ценой
2. **User Contract** - индивидуальные данные каждого пользователя  
3. **Jetton Contract** - стандартный токен

### Функциональность
- Прием TON инвестиций
- Выдача jetton по текущей цене
- Refund по исходной цене
- Админ панель для управления ценой

## ПРОСТАЯ АРХИТЕКТУРА

```
Master Contract (цена, логика) 
    ↓
User Contracts (индивидуальные данные)
    ↓  
Jetton Standard (токены)
```

## SUCCESS CRITERIA

✅ Пользователь может инвестировать TON → получить jetton  
✅ Пользователь может сделать refund по исходной цене  
✅ Админ может управлять ценой  
✅ Система масштабируется через индивидуальные контракты  
✅ Все работает на TON blockchain
